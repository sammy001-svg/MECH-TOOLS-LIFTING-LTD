# MECH-TOOLS LIFTING LTD

A premium, responsive website for MECH-TOOLS LIFTING LTD, specializing in heavy lifting, vehicle recovery, and industrial logistics.

## Features

- **Modern Hero Carousel**: Smooth animations, auto-play, and mobile-optimized content.
- **Service Highlights**: Detailed breakdown of lifting, recovery, and logistics solutions.
- **Project Gallery**: Filterable showcase of past industrial projects.
- **Responsive Navigation**: Streamlined mobile menu and navigation experience.
- **Contact & Quotes**: Dynamic quote request system and integrated contact information.

## Tech Stack

- **Frontend**: Vite, Vanilla JavaScript
- **Styling**: Vanilla CSS (Custom design system)
- **Deployment**: Integrated for GitHub hosting

## Getting Started

1. Clone the repository: `git clone https://github.com/sammy001-svg/MECH-TOOLS-LIFTING-LTD.git`
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
